## Описание проекта

**YaMDb** — это API-сервис для сбора пользовательских отзывов на различные произведения: фильмы, книги, музыку и другие категории контента. При этом сами произведения в системе не хранятся — сервис предназначен исключительно для работы с отзывами, оценками и обсуждениями.

## Основные возможности

* Регистрация пользователей с подтверждением по email
* Аутентификация с использованием JWT-токенов
* Гибкая система ролей и прав доступа:

  * аноним
  * пользователь
  * модератор
  * администратор
* Управление категориями, жанрами и произведениями
* Добавление отзывов с оценкой от 1 до 10
* Комментирование отзывов
* Формирование рейтинга произведений (средняя оценка)
* Поиск и фильтрация по различным параметрам

## Технологии

* Python 3.12
* Django 5.1.1
* Django REST Framework 3.15.2
* Simple JWT
* SQLite3 (используется на этапе разработки)

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/Selitaev-Bogdan/api-yamdb.git
cd api-yamdb
```

### 2. Создание виртуального окружения

```bash
python -m venv venv
```

Активация:

```bash
# Windows
source venv/Scripts/activate

# Linux / MacOS
source venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Применение миграций

```bash
cd api_yamdb
python manage.py makemigrations
python manage.py migrate
```

### 5. Импорт данных (опционально)

```bash
python import_csv.py
```

### 6. Создание суперпользователя

```bash
python manage.py createsuperuser
```

### 7. Запуск сервера

```bash
python manage.py runserver
```

## API эндпоинты

| Эндпоинт                                                             | Метод              | Описание                        | Доступ                                             |
| -------------------------------------------------------------------- | ------------------ | ------------------------------- | -------------------------------------------------- |
| /api/v1/auth/signup/                                                 | POST               | Регистрация пользователя        | Все                                                |
| /api/v1/auth/token/                                                  | POST               | Получение JWT токена            | Все                                                |
| /api/v1/users/                                                       | GET, POST          | Список пользователей / создание | Администратор                                      |
| /api/v1/users/me/                                                    | GET, PATCH         | Профиль текущего пользователя   | Авторизованный                                     |
| /api/v1/users/{username}/                                            | GET, PATCH, DELETE | Управление пользователем        | Администратор                                      |
| /api/v1/categories/                                                  | GET, POST          | Список категорий / создание     | GET: все, POST: администратор                      |
| /api/v1/categories/{slug}/                                           | DELETE             | Удаление категории              | Администратор                                      |
| /api/v1/genres/                                                      | GET, POST          | Список жанров / создание        | GET: все, POST: администратор                      |
| /api/v1/genres/{slug}/                                               | DELETE             | Удаление жанра                  | Администратор                                      |
| /api/v1/titles/                                                      | GET, POST          | Список произведений / создание  | GET: все, POST: администратор                      |
| /api/v1/titles/{id}/                                                 | GET, PATCH, DELETE | Работа с произведением          | GET: все, остальное: администратор                 |
| /api/v1/titles/{title_id}/reviews/                                   | GET, POST          | Отзывы на произведение          | GET: все, POST: авторизованный                     |
| /api/v1/titles/{title_id}/reviews/{review_id}/                       | GET, PATCH, DELETE | Отзыв                           | GET: все, остальное: автор/модератор/администратор |
| /api/v1/titles/{title_id}/reviews/{review_id}/comments/              | GET, POST          | Комментарии к отзыву            | GET: все, POST: авторизованный                     |
| /api/v1/titles/{title_id}/reviews/{review_id}/comments/{comment_id}/ | GET, PATCH, DELETE | Комментарий                     | GET: все, остальное: автор/модератор/администратор |

## Роли пользователей

| Роль          | Права                                                        |
| ------------- | ------------------------------------------------------------ |
| Аноним        | Просмотр произведений, отзывов и комментариев                |
| Пользователь  | Создание отзывов и комментариев, редактирование своих данных |
| Модератор     | Права пользователя + удаление любых отзывов и комментариев   |
| Администратор | Полный доступ ко всем ресурсам и управлению системой         |
